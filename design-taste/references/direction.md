# Design direction (subskill of design-taste)

Load this when starting any design or when the current one reads as generic. Committing to a direction is the single highest-leverage move — it decides every token downstream, so nothing later has to be guessed.

## Why direction first

Without a direction you default: neutral gray, system font, centered hero, three cards. Those defaults *are* the slop. A direction is a constraint that makes choices for you — once you say "editorial," you already know the type wants serifs, generous measure, and asymmetry.

## Infer it from three questions

1. **What is this?** Product marketing, docs, dashboard, portfolio, storefront — the archetype sets density and rhythm (marketing = loose; dashboard = tight).
2. **Who's it for?** Developers reward precision and restraint; consumers reward warmth and imagery; enterprise rewards clarity and trust.
3. **What should it feel like in the first second?** The emotional register — calm, urgent, premium, friendly, serious, experimental.

## Commit to one direction

Pick one and name it. The named directions, their fit signals, and starter tokens live in [`catalog.md`](./catalog.md) — recommend from it or let the user choose. This file is about *how to commit* once you're there.

## Lock three artifacts before building

- **2–3 real reference sites** in this direction (Linear, Stripe, Vercel, Notion, a print magazine — name them). You're matching a real visual language, not inventing from zero.
- **An adjective triad** — three words the design must earn, e.g. "precise, quiet, engineered." Every later choice gets tested against them.
- **The one hero** — what the eye must hit first. Everything else is subordinate.

## Anti-patterns

- Starting from a component library's defaults and calling the result a "direction." That's the absence of one.
- Hedging between two directions — "modern but also playful but also premium." Pick one; a design that wants to be everything reads as nothing.
- Copying a reference's surface (its exact blue) instead of its *system* (why it uses one restrained accent). Steal the logic, not the hex.

Once the direction, references, and triad are locked, the type/color/space systems follow from them — not from defaults.
